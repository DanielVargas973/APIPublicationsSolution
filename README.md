# APIPublicationsSolution
API services container project for the application requested by Julius2Grow


#Puntos a tener en cuenta para desplegar los servicios 

* Se debe tener un equipo que tenga instalado .NET Core versión 3.1
* También se debe tener instalada la base de datos especificada en el correo electrónico enviado
* Los servicios para crear, listar y eliminar post, poseen la configuración de autenticación solicitad (mediante JWT), debe consumir primero el servicio que le permite realizar un registro y luego el servicio 
  que inicia la sesión.
